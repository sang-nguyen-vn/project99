Project Introduction

  • Deploy the tmyapp02.sangnguyen.win web app with a design based on the GitOps model.
  • For CI:
    • Using GitHub as the source code repository.
    • Utilizing GitHub Actions for automated testing and image building.
    • Storing application images on DockerHub.
  • For CD:
    • Automatically deploying the K3s Cluster on AWS using Ansible, Terraform, and Bash scripts.
    • Using ArgoCD and Helm Chart for GitOps. ArgoCD automatically syncs application code and deploys it on the K3s Cluster.
    • Implementing MetalLB as the load balancer to provide IP addresses for Ingress services.
    • Configuring HA Proxy for SSL termination.
    • Managing certificates automatically with Cert-manager.
