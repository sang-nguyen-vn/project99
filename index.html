Project Introduction

Deploy the project99 application with a design based on the GitOps model. The tech stack includes:

  • CI: using GitHub as the source code repository, GitHub Action for automated testing and building images.
  • DockerHub: Storing application images.
  • CD: Using k3s Cluster on AWS automatically deployed via Ansible and Terraform.
  • ArgoCD Helm Chart for deploying the application.
  • MetalLB as the load balancer, HA Proxy as the forwarder.
  • Cert-manager for automatic certificate management.
